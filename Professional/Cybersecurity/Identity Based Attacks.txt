Identity-Based Attack Compendium
Ranked by Estimated Prevalence (High to Low)
============================================

[1] Phishing (Credential Harvesting)
	--------------------------------------------------------------------
	Occurrence Level: CRITICAL (Highest recorded incidents - 40-50% of breaches)
	Attack Technique: 
	- Email-based phishing campaigns with fake login pages
	- Spear phishing targeting specific individuals
	- Clone phishing (legitimate email replicas)
	- SMS phishing (smishing) for MFA codes
	- Voice phishing (vishing) for credential harvesting
	- QR code phishing redirecting to credential theft sites
	- Adversary-in-the-Middle (AitM) phishing bypassing MFA
	
	Primary Traces:
	- Suspicious email headers (mismatched sender domains)
	- URLs with slight misspellings or unusual TLDs
	- Login attempts from unusual geographic locations
	- Failed authentication attempts followed by successful ones
	- Multiple accounts accessed from same IP address
	- Browser user-agent strings inconsistent with user profile
	- Session cookies transmitted to unauthorized domains
	- Proxy/VPN/TOR exit node connections
	- Incomplete SSL certificate chains on phishing sites
	--------------------------------------------------------------------

[2] Password spraying
	--------------------------------------------------------------------
	Occurrence Level: VERY HIGH (25-35% of identity attacks)

	Attack Techniques:
	- Attempting common passwords across many accounts
	- Using seasonal passwords (Winter2024!, Summer2024!)
	- Leveraging leaked password lists
	- Slow, distributed attempts to avoid lockout thresholds
	- Targeting accounts without MFA enabled
	- Using residential proxy networks to distribute attempts

	Identifiable Traces:
	- Multiple failed logins across different accounts from same IP
	- Failed login attempts using common password patterns
	- Distributed failed attempts from multiple IPs in short timeframe
	- Authentication attempts outside business hours
	- Login attempts to disabled/dormant accounts
	- Unusual spike in failed authentication events
	- Geographically distributed login attempts in impossible timeframes
	- Authentication requests with identical user-agent strings
	--------------------------------------------------------------------

[3] Credential Stuffing
	--------------------------------------------------------------------
	Occurrence Level: VERY HIGH (20-30% of identity attacks)

	Attack Techniques:
	- Automated testing of username/password combinations from breaches
	- Using botnets to distribute login attempts
	- Leveraging credential combo lists from dark web
	- Rotating through proxy services to evade detection
	- Testing credentials across multiple services simultaneously

	Identifiable Traces:
	- High volume of failed login attempts in short duration
	- Login attempts using known compromised credentials
	- Unusual user-agent strings or headless browser signatures
	- Multiple accounts accessed successfully from single IP
	- Automation tool fingerprints (Selenium, Puppeteer indicators)
	- Consistent timing patterns in login attempts
	- Perfect username/password pairs submitted without corrections
	- Traffic from data center IPs rather than residential
	- Lack of typical user behavior (no mouse movements, form interactions)
	--------------------------------------------------------------------

[4] Session Hijacking / Sidejacking
	--------------------------------------------------------------------
	Occurrence Level: HIGH (15-25% of identity attacks)

	Attack Techniques:
	- Stealing session cookies via XSS attacks
	- Man-in-the-Middle attacks on unencrypted connections
	- Malware extracting tokens from browser storage
	- Session fixation attacks
	- Token replay attacks
	- OAuth token theft
	- JWT token manipulation
	- Pass-the-Cookie attacks

	Identifiable Traces:
	- Multiple concurrent sessions from different locations
	- Session cookies used from unexpected IP addresses
	- Tokens used beyond normal expiration times
	- Sudden change in user-agent mid-session
	- Session activity from impossible geographic locations
	- Cookie reuse after user logout
	- Modified or tampered JWT payloads
	- Access tokens used without corresponding refresh token activity
	- Session resumption without proper authentication flow
	--------------------------------------------------------------------

[5] Brute Force Attacks (Password Guessing)
	--------------------------------------------------------------------
	Occurrence Level: HIGH (10-20% of identity attacks)

	Attack Techniques:
	- Systematic password guessing against single account
	- Dictionary attacks using wordlists
	- Hybrid attacks combining dictionary and character substitution
	- Reverse brute force (one password, many usernames)
	- Distributed brute force using botnets

	Identifiable Traces:
	- Massive volume of failed login attempts to single account
	- Sequential or patterned password attempts
	- Continuous authentication requests without delays
	- Account lockouts triggered repeatedly
	- Failed attempts using incremental password variations
	- Attacks originating from known malicious IP ranges
	- Unusual spike in authentication traffic
	- Attempts continuing despite lockout mechanisms
	--------------------------------------------------------------------

[6] Pass-the-Hash / Pass-the-Password
	--------------------------------------------------------------------
	Occurrence Level: MODERATE-HIGH (8-15% of advanced attacks)

	Attack Techniques:
	- Extracting NTLM hashes from memory (LSASS)
	- Kerberos ticket extraction and replay
	- Using tools like Mimikatz, Rubeus
	- Golden/Silver Ticket attacks
	- Over-pass-the-hash (pass-the-key)
	- DCSync attacks to harvest credentials

	Identifiable Traces:
	- LSASS process access by unusual processes
	- Kerberos TGT/TGS requests without corresponding authentication
	- Unusual privilege escalation events
	- Lateral movement patterns using stolen credentials
	- Authentication using NTLM when Kerberos is expected
	- Service accounts logging in from workstations
	- Multiple account authentications from single endpoint
	- Golden ticket indicators (unusual ticket lifetimes/domains)
	- Event ID 4624 (logon) with type 9 (NewCredentials)
	--------------------------------------------------------------------

[7] Kerberos-Based Attacks (e.g., Golden Ticket, Pass-the-Ticket)
	--------------------------------------------------------------------
	Occurrence Level: MODERATE (5-12% of domain attacks)

	Attack Techniques:
	- Requesting service tickets for SPNs
	- Offline cracking of encrypted ticket portions
	- Targeting accounts with weak passwords
	- AS-REP roasting (accounts without Kerberos pre-auth)
	- Using tools like Rubeus, Impacket
	- Forging Kerberos TGT with stolen KRBTGT hash (Golden)
	- Forging service tickets with service account hash (Silver)
	- Maintaining persistent domain access
	- Bypassing authentication entirely	

	Identifiable Traces:
	- Unusual volume of TGS requests from single account
	- Service ticket requests for SPNs not typically accessed
	- TGS requests for administrative service accounts
	- RC4 encryption in TGS requests (downgrade attacks)
	- Event ID 4769 with unusual patterns
	- Service ticket requests outside business hours
	- Multiple SPN enumeration activities
	- Tickets requested but never used
	- Kerberos tickets with unusual lifetimes (10+ years)
	- TGT requests without corresponding AS-REQ
	- Service access without TGS requests
	- Tickets for non-existent users
	- Event ID 4769 anomalies (encryption type, ticket options)
	- Authentication from unexpected sources
	- Tickets with suspicious PAC data
	- Domain controller replication without proper authorization	
	--------------------------------------------------------------------

[8] ACCOUNT MANIPULATION/PRIVILEGE ESCALATION
	--------------------------------------------------------------------
	Occurrence Level: MODERATE (5-10% of identity attacks)

	Attack Techniques:
	- Adding users to privileged groups
	- Modifying account permissions
	- Creating backdoor admin accounts
	- SSH key injection
	- Sudo/sudoers file manipulation
	- UAC bypass techniques
	- Exploiting misconfigurations in RBAC

	Identifiable Traces:
	- Unexpected group membership changes (Event ID 4728, 4732)
	- New user account creation outside normal processes (Event ID 4720)
	- Permission/ACL modifications on sensitive objects
	- Sudo command usage anomalies in Linux logs
	- Unauthorized SSH key additions to authorized_keys
	- Changes to sudoers file
	- Registry modifications for privilege escalation (Windows)
	- Scheduled task creation with elevated privileges
	--------------------------------------------------------------------

[9] MFA bypass (phishing, push bombing, proxying)
	--------------------------------------------------------------------
	Occurrence Level: MODERATE (3-7% of MFA-enabled environments)

	Attack Techniques:
	- Sending repeated MFA push notifications
	- Exploiting user fatigue to gain approval
	- Targeting off-hours when users are less alert
	- Social engineering combined with MFA prompts
	- SIM swapping to intercept SMS codes

	Identifiable Traces:
	- Multiple MFA denials followed by approval
	- MFA approvals during unusual hours
	- Rapid succession of MFA requests (bombing pattern)
	- MFA approval from unusual location
	- SIM swap activity on phone carrier
	- Login immediately following MFA approval
	- User reports of unexpected MFA prompts
	- Pattern of MFA requests then calls (social engineering)
	--------------------------------------------------------------------

[10] PASS-THE-CERTIFICATE
	--------------------------------------------------------------------
	Occurrence Level: LOW-MODERATE (2-5% of advanced attacks)

	Attack Techniques:
	- Stealing user/computer certificates
	- Using certificates for authentication bypass
	- Exploiting certificate-based authentication
	- ADCS exploitation (Certifried, ESC attacks)

	Identifiable Traces:
	- Certificate requests from unusual sources
	- Certificate exports from certificate store
	- Authentication using unexpected certificates
	- Certificate template abuse
	- Event ID 4886, 4887 (certificate operations)
	- Unusual PKINIT authentication events
	- Certificates issued outside normal enrollment process	
	--------------------------------------------------------------------

[11] LDAP INJECTION/DIRECTORY TRAVERSAL
	--------------------------------------------------------------------------------
	Occurrence Level: LOW (1-3% of web application attacks)

	Attack Techniques:
	- Injecting malicious LDAP queries
	- Bypassing authentication via LDAP manipulation
	- Extracting directory information
	- Anonymous bind exploitation

	Identifiable Traces:
	- Unusual LDAP query patterns in logs
	- LDAP errors indicating injection attempts
	- Successful authentication with malformed input
	- Excessive LDAP queries from single source
	- Anonymous LDAP binds when not expected
	- Directory information disclosure
	- Special characters in authentication fields

	--------------------------------------------------------------------
