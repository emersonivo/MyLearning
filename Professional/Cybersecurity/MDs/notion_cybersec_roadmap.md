# 🛡️ CyberSec — Conhecimento → Conteúdo
### De Técnico de TI para Criador de Conteúdo em Segurança

> **Como usar este projeto no Notion:**
> 1. Crie uma página principal chamada **"🛡️ CyberSec — Conhecimento → Conteúdo"**
> 2. Dentro dela, crie sub-páginas para cada seção abaixo (📋 Inventário, ⚙️ Pipeline, 📅 Cronograma, 💰 Recursos)
> 3. Cole o conteúdo de cada seção na sub-página correspondente
> 4. As tabelas vão colar diretamente — o Notion converte markdown em databases
> 5. No Inventário, mude o "Status" à medida que revisar cada conceito

---

**Visão Geral do Sistema:**
- 🎯 **Fase 1 (Sem. 1–8):** Revisar Tier 1 → Publicar primeiros conteúdos → Meta $500–1500/mês
- 📖 **Fase 2 (Sem. 9–16):** Revisar Tier 2 → Vídeos longos + primeiro produto pago → Meta $1500–3000/mês
- 🚀 **Fase 3 (Sem. 17–24+):** Revisar Tier 3 → Curso + Comunidade paga → Meta $3000–6000+/mês
